## About

나온케어는 독거노인을 위한 AI 기반 선제형 감정 돌봄 서비스입니다.
기존 음성 비서가 호출어를 기다리는 반응형(Reactive) 방식이라면,
나온케어는 AI가 먼저 안부를 건네는 선제형(Proactive) 방식을 지향합니다.

- 서비스 형태 : 안드로이드 네이티브 앱 + 보호자 웹 대시보드
- 핵심 기술 : 온디바이스 음성 처리, 감정 분석, 선제 대화 엔진
- 특허 출원 : 10-2025-0185553 (2025.11.28, 심사 대기 중)
